01 — Getting Started: AWS CLI v2 Simulator

Welcome to the challenge! Use the AWS CLI v2 simulator to explore and solve each task. The simulator itself is not an attack target—treat it as training infrastructure.

  • Discover credentials during the challenge and register them in your local AWS CLI configuration.
  • Use the following commands to record any access key pair you obtain:
aws configure set aws_access_key_id <ACCESS_KEY>
aws configure set aws_secret_access_key <SECRET_KEY>
  • Only a focused subset of AWS commands is implemented. Check availability with the built-in help system:
aws <service> --help
  • Begin by reviewing IAM information and understanding your effective permissions:
aws iam --help
