05 — Leaky Repo (Download)

This repository is provided for local inspection. Retrieve a copy, review it in your own environment, and explore the history at your pace.

Downloads

  • leaky-repo.tar.gz — compressed archive
  • leaky-repo.bundle — Git bundle snapshot
  • README.md — distribution notes

How to use (locally)

# Option A: tarball
tar -xzf leaky-repo.tar.gz
cd leaky-repo

# Option B: git bundle
git clone leaky-repo.bundle leaky-repo
cd leaky-repo

# Inspect the history
git log --oneline
git log -p
